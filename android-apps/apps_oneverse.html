一句聖經OneVerse


Description

This is an app to receive bibile verse daily. It automatically push to the user daily. Each verse can be saved and deleted. Each verse has articled based on it. This is a good app to reminded to keep with god daily.


Privacy Policy:

This app is a standlone app to receive feeds. No personal users data will be collected.

这是一个独立的应用，不会收集任何用户数据。
